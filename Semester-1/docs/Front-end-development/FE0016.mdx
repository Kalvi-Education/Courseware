# CSS Selectors

  

Welcome to the lesson on Front-end web development. This would be a self-paced class which focuses on one of the fundamental topics in CSS - Selectors. In this lesson, we will learn how to select an HTML element using different CSS selectors to style them.

  

Please make sure that you are completing the entire lesson.

## What are your learning goals in this lesson

Goal 1: Understanding what is CSS Selector

Goal 2: Learning the basic CSS Selectors

Goal 3: Learning all the advanced CSS Selectors

  

## What are CSS Selectors?

[MDN](https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Selectors) describes “in CSS that selectors are used to select/target an HTML element and add styling to it. There are a wide variety of CSS selectors available, e.g simple, combination, pseudo, attribute selectors etc.”

  

Get ready for exploring all the different types of selectors.
