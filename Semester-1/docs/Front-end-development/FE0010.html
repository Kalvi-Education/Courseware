Document and website structure

Welcome to the lesson on Front-end web development. This would be a self-paced class which focuses on Document and website structure. In this lesson, we will learn about various semantic and non-semantic HTML elements required to structure a web page

Please make sure that you are completing the entire lesson.

What are your learning goals in this lesson?​

Goal 1: Understanding basic sections of a document and HTML elements required to structure a web page.

Goal 2: Understanding non-semantic wrappers in HTML

Goal 3: Understanding line breaks and horizontal rules In HTML.

Here is an interesting MDN which will help you understand Document and website structure.

Road map for better learning

Milestone 1: Complete reading the whole document (the time required to read the document is 25-30 minutes) and then continue with the practices which are given below.

Milestone 2: Here are a couple of videos on Document and website structure.
