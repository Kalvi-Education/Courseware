Welcome to the lesson on Front-end web development. This would be a self-paced class which focuses on one of the fundamental topics in CSS - Selectors. In this lesson, we will learn how to select an HTML element using different CSS selectors to style them.

Please make sure that you are completing the entire lesson.

Getting started with CSS

What are your learning goals in this lesson​

Goal 1: What is CSS?

Goal 2: Importance of CSS

Goal 3: Adding CSS to HTML

What is CSS?​

CSS is the language we use to style the HTML document and describes how HTML elements should be displayed. It can control the layout of multiple webpages at a time.

Why use CSS?​

GIF link: https://appsilon.com/static/b5fc2ee778ab20854124eef40e6b1287/how-css-works.gif

CSS is used to define styles for the web pages which includes design, layout and variations in display for different devices and screen sizes. The CSS code is saved in .css files. With an external stylesheet file, you can change the look of an entire website by changing just one file.

How to add CSS to HTML document?​

There are three ways to add CSS to HTML document.

  • External Stylesheet

You can link the external CSS file to HTML document using <link> tag which is to be added inside the <head> of the HTML document.

<link rel="stylesheet" href="styles.css">
  • Internal Stylesheet

Click here to learn and practice how to add internal CSS styles

  • Inline styles

Click here to learn and practice how to add inline CSS styles

End-of-the-lesson assessment

Learning Unit: Styling HTML elements

  1. Where is an HTML document is the correct place to refer to an external style sheet.

    a. In the <head> section

    b. In the <body> section

    c. In the <style> section

    d. At the end of the document

Answer: a

  1. Which HTML tag is used to define an internal style sheet?

    a. <head>

    b. <css>

    c. <stylesheet>

    d. <style>

Answer: d

  1. What is the correct HTML for referring to an external stylesheet?

    a. <style src=”mystyle.css”>

    b. <style>style.css</style>

    c. <stylesheet>mystyle.css</stylesheet>

    d. <link rel=”stylesheet” type=”text/css” href=”mystyle.css”>

Answer: d

  1. Which HTML attribute is used to define inline styles?

    a. style

    b. font

    c. class

    d. Styles

Answer: a

  1. Which is the correct CSS syntax?

    a. body:color=black;

    b. body {color:black;}

    c. {body:color=black;}

    d. {body;color:black;}

Answer: b

  1. Which property is used to change the background color?

    a. background_color

    b. color

    c. background-color

    d. bgcolor

Answer: c

Learning Unit: Styling HTML Selectors

  1. How do you add a background color for all <h1> elements?

    a. h1 {background-color=#ffffff;}

    b. h1.all {background-color:#ffffff;}

    c. all.h1 {background-color:#ffffff;}

    d. h1 {background-color:#ffffff;}

Answer: d

  1. What is the correct CSS syntax for making all the <p> elements bold

    a. p {font-weight:bold;}

    b. <p style=”text-size:bold;”>

    c. p {text-size:bold;}

    d. <p style=”font-size:bold;”>

Answer: a

  1. How do you display hyperlinks without an underline?

    a. a {text-decoration:none;}

    b. a {underline:none;}

    c. a {text-decoration:no-underline}

    d. a {decoration:no-underline;}

Answer: a

  1. Which property is used to change the font of an element?

    a. font-style

    b. font-weight

    c. font-family

    d. font-change

Answer: c

  1. In how many ways can CSS be written in?

    a. 1

    b. 2

    c. 3

    d. 4

Answer: c
