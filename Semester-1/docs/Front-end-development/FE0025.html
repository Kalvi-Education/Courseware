HTML FORMS

Welcome to the lesson on Front-end web development. This would be a self-paced class which focuses on one of the fundamental topics in HTML - Forms.

Please make sure that you are completing the entire lesson.

What are your learning goals in this lesson​

Goal 1: Learning about HTML forms.

Goal 2: Completing the practice exercises.

Road map to complete this lesson​

Milestone 1: Learn about HTML forms with the help of the resources provided below. The expected time to complete this section is 20 minutes.

What are Html Forms?​

As per MDN, “Web forms are one of the main points of interaction between a user and a website or application. Forms allow users to enter data, which is generally sent to a web server for processing and storage, or used on the client-side to immediately update the interface in some way.”

A) In HTML form <input type=”text”> is used for -

  1. None
  2. One-line text
  3. One paragraph
  4. Block of text

B) Which of the following is not the form type for adding text?

  1. Submit button
  2. Text area
  3. Text input
  4. Password input

Answer: a

Learning Unit: HTML Forms

C) Choose the correct option.

  1. HTML form elements are used for taking user input
  2. HTML form elements are defined inside <form> tag
