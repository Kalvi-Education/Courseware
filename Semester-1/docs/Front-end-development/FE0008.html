Styling Lists

Welcome to the lesson on Front-end web development. This would be a self-paced class which focuses on one of the fundamental topics in CSS-Styling lists. In this lesson, we will learn how to style lists.

Please make sure that you are completing the entire lesson.

What are your learning goals in this lesson​

Goal 1: Understanding what is List Styling

Goal 2: Learning different ways to style lists.

What is List Styling?​

As per MDN, “Lists behave like any other text for the most part, but there are some CSS properties specific to lists that we need to know about, as well as some best practices to consider.”

Here is an interesting article by MDN which will help you understand Styling Lists better.

Road map for better learning

Milestone 1: Complete reading the whole document (the time required to read the document is 25 minutes) and then continue with the practices which are given below.
