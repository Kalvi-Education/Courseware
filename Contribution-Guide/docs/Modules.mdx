# Modules and how to use them

## Local images

### Steps

1. Create a folder for your course in `/assets`
2. Import the image using the following syntax:

```md
    ![<alt>](assets/<file_name>)
```

### Example

```md
    ![Kalvi Logo](assets/kalvi-logo.png)
```

### Result

![Kalvi Logo](assets/kalvi-logo.png)

## Collapsible

### Steps

1. Use the `<detail>` element to wrap collapsible content.

### Example

```html
<details>
  <summary>Click to expand!</summary>

  This is some content
</details>
```

### Result

<details> 
    <summary>Click to expand!</summary>

    This is some content

</details>

## Docs Viewer

### Steps

1. Import the DocsViewer component at the top of the file using:

```tsx
import DocsViewer from "@site/src/components/DocsViewer";
```

2. Use the component as follows:

```tsx
<DocsViewer toc={toc} src="<Google Docs/ PDF URL>" />
```

### Example

_Passing in a Google Docs/ Slides/ Sheets URL_

```tsx
<DocsViewer
  toc={toc}
  src="https://docs.google.com/document/d/1Gp5MINDu_HL-zOoRL0Di7JbDJpMe9EtnP-a8dmNbemQ/edit"
/>
```

_Passing in a PDF_

```tsx
<DocsViewer
  toc={toc}
  src="https://www.unicef.org/media/122921/file/State%20of%20Learning%20Poverty%202022.pdf"
/>
```

### Result

import DocsViewer from "@site/src/components/DocsViewer";

_Google Docs_

<DocsViewer
  toc={toc}
  src="https://docs.google.com/document/d/1Gp5MINDu_HL-zOoRL0Di7JbDJpMe9EtnP-a8dmNbemQ/edit"
/>

_PDF_

<DocsViewer
  toc={toc}
  src="https://www.unicef.org/media/122921/file/State%20of%20Learning%20Poverty%202022.pdf"
/>

### Notes

Passing in the `toc={toc}` option automatically detects if there is other content on the page. If not, the docs viewer takes up the whole page.

## Linking to other pages

### Steps

1. Create a link with the syntax `[<link text>](<url after /courseware>)`

### Example

```md
[Check out adding submodules](/Contribution-Guide/docs/Adding-Submodules)
```

### Result

[Check out adding submodules](/Contribution-Guide/docs/Adding-Submodules)
