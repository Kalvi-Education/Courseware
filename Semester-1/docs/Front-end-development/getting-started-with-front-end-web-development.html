Getting started with Front-End Web Development

Welcome to the lesson on Front-end web development. This lesson is a self-paced lesson which focuses on setting up a basic web development environment, understanding the basics of files, get to know what is HTML, CSS & JS. Please make sure that you are completing the entire lesson.

About this course‚Äã

  • Course duration: 21 weeks course
  • No. of modules in the course: 7 modules
  • Duration of a module: 3 weeks
  • No. of hours dedicated for this course per week: 15 hours

This course will not be like a traditional course where teacher/mentor teaches the concepts to the students, instead this course will involve a lot of self-paced lessons. The reason why this course is designed with more focus on self-paced learning, is because we at kalvi and a lot of tech experts beleive that this is a best way to learn Front-end web develoment.

This course is made up of different modes of lessons, here are the different modes and its details.

  • Lecture: This is a live lesson in which Kalvi's Mentor will be teaching the lesson.
  • Self-paced lesson: This is a self-paced lesson where studnets will be learning and practicing the concept in Kalvi's learning platform.
  • Lab: This is a self-paced lesson where you will work on practically implementing some of the most exciting concepts you have been learning in Front-end Web Development.
  • Code-along: Code-along is a live class lesson in which you will code with the mentor to practically apply the concepts learnt in the previous studies in a project.
  • TSI (Teacher Student interactions): TSI will be an interactive live season, where teachers and students will be discussing crucial technical interview questions and go deeper into the concepts learnt previously by using Inquiry-based learning.
  • Module Assessment: This is a project-based assessment which will be happening at the end of every module, where you will be assessed based on your project.

What is Front End?‚Äã

The fact that we could create this course and you could learn it online is not all easy. We all owe it to one person, Sir Tim Berners Lee, the brain behind the World Wide Web (www). He also created HTML and HTTP along with the first web page editor, the browser.

All these technologies helped him create the first website ever. Sounds interesting! So you see, that's how technology evolved.

And today, every website has two parts to it - Front End and Back End. You can probably visualize them this way,

Understanding & working on both FrontEnd & BackEnd together is a little overwhelming. So, in this semester let's first focus on Front-End for the next few months. Front-end development involves building web pages and user interfaces for web applications.

A Front-end developer implements the structure, design, behaviour, and animation of everything you see on the screen when you open up web applications or mobile apps.

Frontend development stands firmly on three pillars: HTML for markup, CSS for styling and JavaScript for logic and interactions.

Source: Giphy

Creating beautiful webpages with HTML, CSS, and JavaScript would precisely compose the frontend of your web applications.

Installing VS Code‚Äã

VS Code includes great tooling for web technologies such as JSX/React, HTML, CSS, SCSS, Less, and JSON. Throughout the course, we'll be using the VS code extensively. It is not just an editor, the features supported by VS Code reduce the tasks of any developer significantly. Along with VS code, we will be installing a few additional extensions too.

System requirements for the VS Code:

Visual Studio Code is a small download (< 100 MB) and has a disk footprint of 200 MB. VS Code is lightweight and should easily run on today's hardware.

How to install VS Code?

  • Open VS Code in your browser using this link - Download Visual Studio Code

  • Now VS code would have detected your OS.

  • Download and install VS Code for your system (follow the instructions for installation)
