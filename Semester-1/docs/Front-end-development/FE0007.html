Fundamental Text and Font Styling

GIF link: https://c.tenor.com/DRsTGW2fl7sAAAAC/smile-font.gif

What are your learning goals in this lesson?​

Goal 1: Understanding what is fundamental text and font styling.

Goal 2: To test comprehension of fundamental CSS theory, syntax, and mechanics

Goal 3: Learning all the advanced text and font styling.

What is involved in styling text in CSS?​

MDN describes “If you have worked with HTML or CSS already, e.g., by working through these tutorials in order, then you know that text inside an element is laid out inside the element's content box. It starts at the top left of the content area (or the top right, in the case of RTL language content), and flows towards the end of the line. Once it reaches the end, it goes down to the next line and flows to the end again. This pattern repeats until all the content has been placed in the box.”

Get ready for exploring all the different types of styling text in CSS.

By now we would have got a very good undertaking on “What is fundamental text and font styling”, let’s explore its types.

Here is an interesting article by freeCodeCamp which will help you understand how to Change Text Color and Size with an HTML Tag.

Fonts​

Choosing the right font has a huge impact on how the readers experience a website. The right font can create a strong identity for your brand. Using a font that is easy to read is important. The font adds value to your text. It is also important to choose the correct color and text size for the font.

  • Click here to learn from an article about CSS fonts. (10 minutes)
  • To learn more, watch this video on CSS fonts (4 minutes)
  • To learn more, watch this video on Web fonts. (10 minutes)

Text Layout​

With CSS, we are able to control the exact alignment, spacing between your characters, words and lines. You now have even more ways to lay out your text for optimum readability and style.

  • To learn more, watch this video on Text-Layout. (12 minutes)

Font shorthand​

Many font properties can also be set through the shorthand property font. These are written in the following order: font-style, font-variant, font-weight, font-stretch, font-size, line-height, and font-family.

  • To learn more, watch this video on CSS Font shorthand (5 minutes)

End-of-the-lesson assessment

Before you go, here’s a quick 5-question quiz for you.

Learning Unit: Text and font styling

  1. Which of the following property sets the size of the font?

    a. font-size

    b. font-vari

    c. font-styl

    d. font-weig

Answer: a

  1. These give the choice of ‘Regular’, ‘Italic’, ‘Bold’, or ‘Bold Italic’ for the style of font.

    a. font-size

    b. font-style

    c. ribbons

    d. None of the above

Answer: b

  1. Which of the following is a CSS property?

    a. background-color:

    b. body

    c. lightblue;

    d. { }

Answer: a

  1. What does CSS stand for?

    a. Creative Style Sheets

    b. Cascading Style Sheets

    c. Colorful Style Spread

    d. Computer Style Spread

Answer: b

  1. What symbols go around the properties for each CSS selector?

    a. < >

    b. [ ]

    c. { }

    d. ( )

Answer: c

Learning Unit: Fonts

  1. Will this CSS code work?

    font-family: Arial, Helvetica, sans-serif;

    a. Yes

    b. No

Answer: a

  1. What font will print with this CSS styling?

    Font-family: "Helvetica", "Avant Garde", "Arial", sans-serif;

    a. sans-serif

    b. Avant Garde

    c. Arial

    d. Helvetica

Answer: d

  1. How can you tell if a font is serif or sans-serif?

    a. It is serif if it has the small line at the end of the letter.

    b. It is sans-serif if it has the small line at the end of the letter.

    c. You can't tell by looking, you have to look it up

    d. All fonts can be defined as serif or sans-serif

Answer: a

  1. Which of the following is used to associate a font name to be used in a style sheet with some downloadable font?

    a. @font-face

    b. @charset

    c. @media

    d. !important

Answer: a

  1. Which of the following property sets the style of a font?

    a. font

    b. style

    c. font-style

    d. @font-face

Answer: c
