Welcome to the lesson on Front-end web development. This would be a self-paced class that focuses on one of the fundamental topics in CSS - Overflowing content. In this lesson, we will learn how to select an HTML element using different CSS selectors to style them.

Please make sure that you are completing the entire lesson.

What is Overflow?​

GIF source: Drink Spilling GIF by The Tenderloins - Find & Share on GIPHY

MDN describes overflow as “Everything in CSS is a box. You can constrain the size of these boxes by assigning values of width and height (or inline-size and block-size). Overflow happens when there is too much content to fit in a box. CSS provides various tools to manage overflow.”

  2. How to add a scroll bar.
  3. How to control overflow in x and y directions independently.
