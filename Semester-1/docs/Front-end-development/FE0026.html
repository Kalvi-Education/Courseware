Style Media and Forms

Welcome to the lesson on Front-end web development. This would be a self-paced class that focuses primarily on how to style media and forms in CSS.

Please make sure that you are completing the entire lesson.

What are your learning goals in this lesson​

Goal 1: Learning how to style media elements

Goal 2: Learning how to style form elements

Goal 3: Completing the practice exercise.

According to MDN, “Images, other media, and form elements behave a little differently from regular boxes in terms of your ability to style them with CSS.”

Road map to complete this lesson​

Milestone 1: Learn how to style media elements in CSS with the help of the resources provided below. The expected time to complete this section is 25minutes.

What are replaced elements in CSS?​

As per MDN, “Images and video are described as replaced elements. This means that CSS cannot affect the internal layout of these elements — only their position on the page amongst other elements.”

Sizing Images​

Everything in CSS generates a box. If you place an image inside a box that is smaller or larger than the intrinsic dimensions of the image file in either direction, it will either appear smaller than the box or overflow the box.
