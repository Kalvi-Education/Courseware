DEBUGGING CSS

Welcome to the lesson on Front-end web development. This would be a self-paced class which focuses on Debugging in CSS

Please make sure that you are completing the entire lesson.

What are your learning goals in this lesson​

Goal 1: Learning about debugging in CSS

Goal 2: Learn how to inspect CSS and edit/add CSS properties.

Road map to complete this lesson​

Milestone 1: Learn about debugging in CSS and dev tools which help us debug CSS codes.

What is Debugging in CSS?​

As per MDN, “Sometimes when writing CSS you will encounter an issue where your CSS doesn't seem to be doing what you expect. Perhaps you believe that a certain selector should match an element, but nothing happens, or a box is a different size than you expected.”

End-of-the-lesson assessment

A) Which of the following CSS property, if added to the below-mentioned code will show desired output?

Code:

span{

width : 20px;

height : 20px;

}

  1. display : flex;
  2. display : inline-block;
  3. display : flexbox;
  4. None of these

Answer: Option 2

Learning Unit: Debugging in CSS.

  1. blue;
  2. black;
  3. green;
  4. None of these

Answer: Option 3

Learning Unit: Debugging in CSS.
