Backgrounds & borders

Welcome to the lesson on Front-end web development. This would be a self-paced class which focuses on one of the fundamental topics in Basics of HTML & CSS. In this lesson, we will learn how to style Backgrounds & borders in CSS.

Please make sure that you are completing the entire lesson.

What are your learning goals in this lesson?​

Goal 1: Understanding what is CSS Backgrounds & borders

Goal 2: Learning how to style the background and border of boxes.

Goal 3: Learning all the advanced concepts of CSS Backgrounds & borders

What are Backgrounds & borders in CSS?​

MDN describes Backgrounds & borders in CSS as “Styles in the CSS Backgrounds and Borders module allow filling backgrounds with color or an image (clipped or resized) or modifying them in other ways. These styles can also decorate borders with lines or images, and make them square or rounded.”

Get ready for exploring all the different types of Backgrounds & borders in CSS.

End-of-the-lesson assessment

Learning Unit: Backgrounds

A) Which property can be used to give a background color to an element?

  1. background-color
  2. backgrounds-color
  3. back-color
  4. None of these

Answer: Option 1

B) Which of the following is the correct syntax to make the background-color of all paragraph elements to yellow?

  1. p {background-color : yellow;}
  2. p {background-color : #yellow;}
  3. all {background-color : yellow;}
  4. all p {background-color : #yellow;}

Answer: Option 1

C) Which property and value are used to add a background without any repetitions?

  1. Background-size:cover;
  2. Background-size:full;
  3. Background-size:no-repeat
  4. None of these

Answer: Option 1

D) Which of these is not a value for background-repeat property?

  1. x-repeat
  2. no-repeat
  3. repeat-y
  4. None of the above

Answer: Option 1

E) Which of these is the correct representation of the background shorthand property-

  1. backpart
  2. background-short
  3. background-shorthand
  4. background

Answer: Option 4

Learning Unit: Borders

F) The CSS property used to make the rounded borders, or rounded corners around an element are -

  1. border-collapse
  2. border-radius
  3. border-spacing
  4. None of the above

Answer: Option 2

G) Which of the following property defines the border-width, border-style and border-color of an element’s top border in a single declaration?

  1. border
  2. border-top
  3. Border-top-corner
  4. border-top-style

Answer: Option 2

H) Which of the following specifies the distance between the borders of adjacent cells?

  1. border-spacing-cell
  2. border-width-spacing
  3. border-spacing
  4. cell-spacing

Answer: Option 3

I) Which of the following CSS property sets the color of an element’s border?

  1. border-background
  2. border-background-color
  3. border-color
  4. all of the mentioned

Answer: c

J) Identify the CSS property defining bottom-left corner shape of the border?

  1. border-radius
  2. border-corner-radius
  3. border-bottom-left-radius
  4. border-left-radius

Answer: Option 3
